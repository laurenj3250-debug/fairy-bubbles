# Residency Tracker - Mobile-First Redesign

## Design Goals

1. **2-tap minimum flow** - Mood + Quit/Stay = logged
2. **Large tap targets** - 64px emoji buttons, 56px action buttons
3. **Sticky confounders** - Set once in morning, auto-apply all day
4. **Beautiful UI** - Gradients, subtle shadows, smooth animations
5. **Fixed submit** - Always visible at bottom of viewport

## Visual Design

### Colors
- Background: Dark slate (`bg-slate-900`)
- Cards: Glass morphism (`bg-slate-800/50 backdrop-blur`)
- Quit: Red gradient (`from-red-600 to-red-700`)
- Stay: Green gradient (`from-emerald-600 to-emerald-700`)
- Primary action: Blue gradient (`from-blue-600 to-blue-500`)
- Selected state: Ring + scale + glow

### Animations
- Mood select: Scale up 1.1 + glow pulse
- Button press: Scale down 0.95
- View transitions: Slide + fade
- Success: Confetti burst or checkmark animation

### Typography
- Headers: Bold, tracking-tight
- Body: text-slate-300
- Muted: text-slate-500
