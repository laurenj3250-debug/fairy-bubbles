# Item Sprites

Place item/loot sprite images here.

## Naming Convention
- Use lowercase with hyphens: `item-name.png`
- Example: `snack.png`, `basic-net.png`

## Current Items Needed

### Common Items
- `snack.png` - Restores 2 HP (food/treat)
- `basic-net.png` - +2 capture chance (simple net)
- `wisdom-charm.png` - +1 WIS charm (amulet/trinket)
- `guard-brace.png` - +1 DR defense item (bracelet/armor)

### Uncommon Items
- `dexterity-charm.png` - +1 DEX charm (agility trinket)
- `evasion-cloak.png` - +10% dodge (cloak/cape)

### Rare Items
- `rare-net.png` - +4 capture chance (ornate net)
- `evolution-gear.png` - Used for evolution (magical gear/stone)

## Recommended Size
- **32x32 or 48x48 pixels** (icon size)
- Transparent background (PNG)
- Pixel art or clean icon style

## Icon Design Tips
- Make them distinct and recognizable
- Use color coding for rarity:
  - Common: White/Gray
  - Uncommon: Green
  - Rare: Blue/Purple
