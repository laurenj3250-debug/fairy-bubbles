# Biome Background Sprites

Place biome/world background images here.

## Naming Convention
- Use lowercase with hyphens: `biome-name.png`
- Example: `biome-garden.png`, `biome-river.png`

## Current Biomes Needed

1. **`biome-garden.png`** - Garden Glade
   - A peaceful meadow where beginners find their footing
   - Soft greens, flowers, sunny atmosphere
   - Unlocked: Always available

2. **`biome-river.png`** - Moonlit River
   - A mystical waterway flowing with moon-blessed currents
   - Blues, purples, moonlight reflections
   - Unlocked: Player Level 3

3. **`biome-ember.png`** - Ember Orchard
   - An ancient grove where fruit trees burn with eternal flame
   - Warm oranges, reds, glowing embers
   - Unlocked: Player Level 5

4. **`biome-dunes.png`** - Zephyr Dunes
   - Windswept sands hide swift and cunning creatures
   - Sandy yellows, wind effects, desert vibes
   - Unlocked: Player Level 7

5. **`biome-clockwork.png`** - Clockwork Garden
   - A mechanical paradise where time itself bends to habit
   - Steampunk, gears, clockwork flowers
   - Unlocked: Player Level 9

## Recommended Size
- **1920x1080** or similar 16:9 ratio
- Can be larger for parallax scrolling
- PNG or JPEG format
