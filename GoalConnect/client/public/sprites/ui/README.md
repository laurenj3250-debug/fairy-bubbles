## Recommended Sizes
- **Icons**: 24x24 or 32x32 pixels
- **Buttons**: 48x48 or 64x64 pixels
- **Borders**: Variable, depends on card size

## Design Style
- Consistent with overall game aesthetic
- Clear and readable at small sizes
- Transparent backgrounds (PNG)
