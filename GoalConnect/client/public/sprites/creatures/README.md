# Creature Sprites

Place creature sprite images here. Each creature should have a sprite file.

## Naming Convention
- Use lowercase with hyphens: `creature-name.png`
- Example: `mossbun.png`, `teafox.png`, `glowmoth.png`

## Current Creatures Needed

### Garden Glade (Common/Uncommon)
- `mossbun.png` - A fluffy moss-covered rabbit
- `teafox.png` - A fox that smells of chamomile
- `glowmoth.png` - A moth with bioluminescent wings

### Moonlit River (Water creatures)
- `pondskipper.png` - A water-walking lizard
- `lunafish.png` - A fish that swims through air at night

### Ember Orchard (Fire creatures)
- `emberspark.png` - A small fire sprite
- `flamebat.png` - A bat that feeds on burning fruit

### Rare/Epic
- `wonderphant.png` - A porcelain elephant that grants wishes

## Recommended Size
- **64x64 pixels** (pixel art style)
- Transparent background (PNG)
- Crisp, pixelated rendering
