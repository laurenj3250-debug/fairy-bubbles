## Palette
- Background: #140B0D to #1A1014
- Outer shell dark: #3B241F
- Main top face: #6B3F2B
- Warm highlight: #C98A52
- Text primary: #F2DFC8
- Text secondary: #B89476
- Shadow: rgba(0,0,0,0.35) to rgba(0,0,0,0.55)

## Radii
- Large outer cards: 30-36px
- Inner trays: 22-28px
- Pills/nav: 26-32px
- Small chips: 18-22px
- Day cells: 14-18px

## Every component = 3-layer object
1. Outer shell (darker body)
2. Top face (lighter)
3. Shadow base (contact + ambient)
